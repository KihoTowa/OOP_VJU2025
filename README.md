# Những gì có trong repository này
Repository này bao gồm các bài tập về nhà có trong file bài giảng của lớp CSE3011, học phần lập trình hướng đối tượng (OOP)
Tên của các file bài tập được đánh dấu theo tên trong phần ảnh có ở trong slide 
(VD: Trong slide ghi excercise 3.3 #ThinkPython thì sẽ đặt là Excercise 3.3) 
Nếu bài tập không phải #ThinkPython thì sẽ đặt theo tuần mà bài tập đó xuất hiện trong slide
(Ví dụ xuất hiện trong slide tuần 1 thì sẽ đặt là Week1_HelloWorld)
